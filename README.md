# Unlock Your Digital Potential

Discover the best tools and strategies to grow your online business.
In-depth guides, reviews, and resources to help you improve your digital marketing skills.

## Who is this project for?

For digital professionals and workers who want to stay updated and find useful tools for their work.

## What does it offer?

- **Blog with marketing-related articles**
  - Marketing Theory
  - Digital Marketing
  - SEO
  - Marketing Psychology
  - Web Development
  - Legal

- **Tools Section**
  - Recommendations and reviews of the best tools for digital marketing
  - Detailed sheets, pros/cons, and use cases

## Project Stack

- **Astro**: Main framework for static site generation
- **React**: Used for interactive components where needed
- **Tailwind CSS**: For fast and modern styling
- **TypeScript**: Static typing for more reliable code
- **MDX**: Write articles and content in Markdown with interactive components
- **astro-color-scheme**: Light/dark theme management
- **astro-icon**: SVG icon management
- **@astrojs/tailwind**: Tailwind integration in Astro
- **@astrojs/mdx**: MDX support in Astro
- **@astrojs/react**: React support in Astro
- **Other dependencies**: See `package.json` for the full list

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Add Astro integrations:

```bash
npm run astro add
```
